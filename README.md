# Сервис проверки должников и арестов по ИИН

Приложение для проверки информации о должниках, исполнительных производствах и арестах по ИИН на сайте aisoip.adilet.gov.kz.

## Описание

Сервис автоматизирует проверку информации о:

- Исполнительных производствах в реестре должников
- Подробной информации о каждом исполнительном производстве
- Наличии арестов, запретов и других ограничений
- Реквизитах для оплаты задолженностей

## Требования

- Node.js (версия 16+)
- NPM или Yarn

## Установка

1. Клонируйте репозиторий:
```
git clone https://github.com/yourusername/debtor-checker.git
cd debtor-checker
```

2. Установите зависимости:
```
npm install
```

## Запуск сервера

Для запуска сервера в режиме разработки:
```
npm run dev
```

Для запуска в продакшн-режиме:
```
npm start
```

После запуска сервер будет доступен по адресу: http://localhost:3000

## Использование

1. Откройте приложение в браузере
2. Введите 12-значный ИИН в поле ввода
3. Нажмите кнопку "Проверить"
4. Дождитесь завершения проверки и отображения результатов

## Технологии

- Node.js и Express (серверная часть)
- Puppeteer (для взаимодействия с сайтами)
- Bootstrap 5 (интерфейс)
- Vanilla JavaScript (клиентская логика)
